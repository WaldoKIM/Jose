for checking the status

please type;

-git status

then for save the changes

please type;

-git add test,txt

or type;

-git add . 

for all files

then for commit- which means submit,

please type;

-git commit -m "your message as commnetary"


then for check update status

please type;

-git status

for set your remote repository path

please type;

-git remote add origin https://github.com/YourID/YourProjectname.git

e.g.

-git remote add origin https://github.com/WaldoKIM/Jose.git

now you ready to upload to remote repository

for upload your files

please type;

git push -u origin master

well done!
